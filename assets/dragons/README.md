# Картинки драконов

Просто положи сюда файлы по схеме:

    {ключ_вида}_{стадия}.{jpg|jpeg|png|webp}

Ключи видов — это ключи из `SPECIES` в `game_data.py`, например:
`fire_common`, `fire_rare`, `shadow_mythic` и т.д. (полный список — 48 штук,
смотри драконы_таблица.txt).

Стадии — `baby` и `adult` (можно не делать `teen`, бот сам подставит
картинку взрослого, а если её тоже нет — детскую).

Примеры имён файлов:
    fire_common_baby.png
    fire_common_adult.png
    shadow_mythic_baby.jpg
    shadow_mythic_adult.jpg

Если для какого-то дракона картинок нет — бот просто покажет карточку
текстом, как сейчас, ничего не сломается. Можно добавлять картинки
постепенно, по одному виду за раз.
